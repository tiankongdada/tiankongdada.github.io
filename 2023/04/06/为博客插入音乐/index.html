为博客插入音乐

发表于2023-04-06|更新于2023-04-06|博客教程
|字数总计:380|阅读时长:1分钟|阅读量:

前言

偶然间看到别人的博客有音乐挂件，必须安排！

搜了下可以利用插件hexo-tag-aplayer 播放器支持一些主流平台的音乐播放，这里我用了网易云音乐。

效果：可以折叠起来，展开来是这样的：

教程

安装hexo插件

打开Mac终端cd到博客目录

npm install --save hexo-tag-aplayer
配置文件

在Hexo配置文件_config.xml搜索aplayer，搜索不到就在最后添加.

aplayer:
  meting: true
  asset_inject: false

主题配置文件中修改配置

aplayerInject:
  enable: true
  per_page: true

还有个配置修改，作用时切换页面时音乐不会中断

pjax:
  enable: true
  exclude:
插入Aplayer播放器
引入播放器的位置和对应的单个音乐或者歌单的设置
inject:
  head:
  bottom:
    # - <script src="xxxx"></script>
     - <div class="aplayer" data-id="8283046977" data-server="netease" data-type="playlist" data-fixed="true" data-mini="true" data-listFolded="false" data-order="random" data-preload="none" data-autoplay="true" muted></div>

div内的有必要修改的属性对应如下:

选项 描述
data-id 歌曲id/歌单id
data-server 平台：netease(网易云音乐),tencent(腾讯音乐),kugou(酷狗)
data-type song, playlist (一首，一列歌单)
